MODES

A user can switch modes between work and hire process . He/She can both work and hire employees for a work simultaneously . A user can hire a single person or a group for a project . Hiring process is simple in which a user needs to be in hire mode . Whenever he clicks on the post project button a window open for project details . Once its filled up a list of freelancer is enlisted . After freelancer are chosen and a project is posted a notifcitaion is sent individually to each freelancers and a window with set preset count down starts . Within that count down a worker is needed to respond to the project. After a user accepts a project he/she can have a chat with the employer for futher negotitaion . Final step includes the freelancer hiring the worker for a job with an agreement .
